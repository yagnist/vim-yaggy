# Vim Yaggy Syntax

This is a vim syntax file for [Yaggy][yaggy] scenarios.

The `filetype` will be set to `yaggy` for `*.yg` files.


## Installation

Install using your favorite Vim package manager.

Example using [Plug][plug]:

```
Plug 'yagnist/vim-yaggy'
```
